-------------------------------------------------------------------

add --- 1. It is used to add an argument to the value. Example

{{ Age | add:"2" }}

-------------------------------------------------------------------

-------------------------------------------------------------------

addslashes --- 2. It is used to add slashes before quotes. Useful for escaping strings in CSV. Example

{{ Name | addslashes }}

If value is “I’m sagor ahmed”, the output will be “I\’m sagor ahmed”.

-------------------------------------------------------------------

capfirst --- 3. It is used to capitalize the first character of the value. If the first character is not a letter, this filter has no effect. Example

{{ Name_2 | capfirst }}

-------------------------------------------------------------------

center --- 4. It is used to center the value in a field of a given width. Example

{{ Name_2 | center:"15" }}

If value is “sagor ahmed”, the output will be ” sagor ahmed “.

-------------------------------------------------------------------

-------------------------------------------------------------------

cut --- 5. It is used to remove all values of arg from the given string. Example

{{ Name_2 | cut:" " }}

If value is “sagor ahmed”, the output will be “sagorahmed”.

-------------------------------------------------------------------

-------------------------------------------------------------------

-------------------------------------------------------------------

date --- 6. It is used to format a date according to the given format. Example

{{ Date__Time | date:"D d M Y" }}

-------------------------------------------------------------------

-------------------------------------------------------------------

time --- 7. It is used to format a time according to the given format. Example

{{ Date__Time | time:"" }}

-------------------------------------------------------------------

-------------------------------------------------------------------

default --- 8. It is used to give a default value to a variable. If variable evaluates to False, it will use the default argument given else the variable value itself. Example

{{ Str | default:"The string is empty." }}

-------------------------------------------------------------------

-------------------------------------------------------------------

dictsort --- 9. It takes a list of dictionaries and returns that list sorted by the key given in the argument. Example

{{ lis | dictsort:"name" }}

-------------------------------------------------------------------

-------------------------------------------------------------------

divisibleby --- 10. It returns True if the value is divisible by the argument. Example

{{ Age | divisibleby:"3" }}

-------------------------------------------------------------------

-------------------------------------------------------------------

first --- 11. It is used to return the first item in a list. Example

{{ lis | first }}

-------------------------------------------------------------------

-------------------------------------------------------------------

last --- 12. It is used to return the last item in a list. Example

{{ lis | last }}

-------------------------------------------------------------------

-------------------------------------------------------------------

join --- 13. It is used to join a list with a string, like Python’s str.join(list) Example

{{ lis | join:" // " }}

-------------------------------------------------------------------

-------------------------------------------------------------------

length --- 14. It is used to return the length of the value. This works for both strings and lists. Example

length of list : {{ lis | length }}

length of string : {{ Name_2 | length }}

-------------------------------------------------------------------

-------------------------------------------------------------------

lower --- 15. It is used to converts a string into all lowercase. Example

{{ Name_3 | lower }}

-------------------------------------------------------------------

-------------------------------------------------------------------

make_list --- 16. It is used to return the value turned into a list. For a string, it’s a list of characters. For an integer, the argument is cast to a string before creating a list. Example

{{ Name_3 | make_list }}

{{ 13237 | make_list }}

-------------------------------------------------------------------

random --- 17. It is used to return a random item from the given list. Example

{{ lis | random }}

-------------------------------------------------------------------

slice --- 18. It is used to return a slice of the list. Example

{{ lis | slice:":2" }}

-------------------------------------------------------------------

slugify --- 19. It is used to convert to ASCII. It converts spaces to hyphens and removes characters that aren’t alphanumerics, underscores, or hyphens. Converts to lowercase. Also strips leading and trailing whitespace. Example

{{ Name_4 | slugify }}

-------------------------------------------------------------------

title --- 20. It is used to convert a string into titlecase by making words start with an uppercase character and the remaining characters lowercase. This filter makes no effort to keep “trivial words” in lowercase. Example

{{ Name_3 | title }}

If value is “sAgOr aHmEd”, the output will be “Sagor Ahmed”

-------------------------------------------------------------------

-------------------------------------------------------------------

upper --- 21. It is used to converts a string into all uppercase. Example

{{ Name_3 | upper }}

-------------------------------------------------------------------

-------------------------------------------------------------------

wordcount --- 22. It is used to return the number of words. Example

{{ Name_4 | wordcount }}
